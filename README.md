# Engauge Digitizer Education Portable Release

This repository contains the education-focused portable release version of Engauge Digitizer.

This build is intended for Windows portable use and includes the application files needed to run without installing Engauge Digitizer separately. It also includes added digitizing automation buttons:

- Auto Axis, for automatic axis calibration support.
- Auto Curve, for automatic curve/data-point population support.

Run the portable application from the release folder with `Start Engauge Digitizer.cmd` or `Engauge.exe`.

Settings are stored locally beside the portable application when `portable-settings.ini` is present.
